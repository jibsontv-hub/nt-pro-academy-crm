WIE BEKOMMST DU DEINE STIMME REIN?

Option A: SELBER AUFNEHMEN (kostenlos, 100% du)
1. iPhone Sprachmemos öffnen
2. Text einsprechen (siehe stepTexts in willkommen.html)
3. Per AirDrop/WhatsApp auf den Mac
4. In MP3 umwandeln (online: cloudconvert.com)
5. Hier in diesen Ordner kopieren

Option B: ELEVENLABS VOICE CLONING ($5-22/Monat)
1. https://elevenlabs.io anmelden
2. Voice Lab → Add Voice → Instant Voice Clone
3. 1-3 Min eigene Stimme hochladen (Mikrofon oder Aufnahme)
4. Voice ID kopieren
5. Mit ihrer API alle Tour-Texte synthetisieren
6. MP3s downloaden, hier ablegen
   → Ergebnis: deine Stimme spricht alle Tour-Texte
   → Klingt 100% wie du, fast nicht zu unterscheiden

Option C: OPENAI TTS ($0.015 pro 1000 Zeichen)
1. https://platform.openai.com — TTS-API
2. Voice "shimmer" oder "nova" nutzen (sehr menschlich)
3. MP3s synthetisieren, hier ablegen

Sobald die Dateien hier liegen, lädt die App sie automatisch.
Wenn nicht, fällt sie auf Browser-Standard-Stimme zurück.
